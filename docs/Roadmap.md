# Roadmap

## PoC Baseline

- Native Android app shell.
- Reown/AppKit WalletConnect connection to Neon Mobile.
- Wallet session-derived chain and account.
- Private-net RPC validation.
- NEP-17 token balance reads for the connected account.

## MVP Direction

- Send one real FORGE write operation through Neon Mobile.
- Port the web FORGE operation payloads into native Android use cases.
- Add transaction result handling and state refresh.
- Add integration documentation for emulator and physical device testing.
- Add Android CI for build, lint, and targeted tests.
- Replace temporary tunnel/RPC configuration with a safer development profile
  mechanism.
